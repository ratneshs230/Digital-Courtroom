üöÄ Getting Started

Step 1: Set Up Your API Key

NyayaSutra uses Google's Gemini AI for intelligent document analysis and courtroom simulation. You'll need at least one API key to get started.

Get a Free API Key

Visit Google AI Studio and sign in with your Google account. Click "Create API Key" to generate a free key.

Add Key to NyayaSutra

In NyayaSutra, look at the bottom of the left sidebar. Click on "API Keys" to expand the panel, then paste your key and press Enter or click the + button.

Add Multiple Keys (Recommended)

For uninterrupted usage, add multiple API keys. Click "Bulk Import" to paste several keys at once (one per line). The system will automatically rotate through them.

üí°
Pro Tip: Create 3-5 API keys from different Google accounts for reliable performance. Each free key has usage limits, but multiple keys provide automatic failover.

Step 2: Create Your First Project

Click "New Case" on Dashboard

From the main dashboard, click the "New Case" button to open the project creation form.

Fill in Case Details

Enter a project name, case title, and brief description. Select whether you're representing the Petitioner or Respondent.

Upload Case Documents

Drag and drop or click to upload relevant documents (PDF, DOCX, TXT). Include FIRs, affidavits, court orders, evidence, and any other relevant materials.

Create Project

Click "Create Project" to start. The AI will begin analyzing your documents automatically.

How Analysis Works

When you create a project, NyayaSutra's AI engine performs a comprehensive analysis of all uploaded documents:

Document Processing

Each document is processed individually to extract text, identify the document type, and understand its legal significance.

Information Extraction

AI extracts key information: dates, parties involved, legal provisions cited, claims made, and evidence referenced.

Perspective Generation

Two comprehensive perspectives are generated - one for Petitioner and one for Respondent - each with complete legal analysis.

Cross-Reference Analysis

The AI identifies disputed facts, contradictions, and connections between different documents.

üîë Managing API Keys

API keys are essential for NyayaSutra's AI features. Managing them properly ensures uninterrupted service.

API Key Manager

Access the API Key Manager by clicking the "API Keys" section at the bottom of the left sidebar.

Key Status Indicators

  • ‚óè Green (Active): Key is working correctly
  • ‚óè Red (Failed): Key hit rate limit or is invalid
  • ‚óè Gray (Unknown): Key hasn't been used yet

Adding Keys

Single Key

  1. Expand the API Keys section
  2. Paste your key in the input field
  3. Press Enter or click the + button

Bulk Import

  1. Click "Bulk Import" tab
  2. Paste multiple keys (one per line)
  3. Click "Import Keys"

Automatic Failover

When a key hits rate limits:

  1. The key is marked as "failed" with a red indicator
  2. The system automatically tries the next available key
  3. Failed keys are retried after a 60-second cooldown
  4. You can manually reset failed keys using the refresh button
üí°
Best Practice: Maintain 3-5 API keys for smooth operation. Free tier keys have limits, so multiple keys ensure continuous availability.

‚ùì Frequently Asked Questions

Why is the AI analysis taking too long?
Analysis time depends on the number and size of documents. Each document is analyzed individually for thorough extraction. For large cases with many documents, this can take several minutes. Ensure you have a stable internet connection and valid API keys.
Can I use NyayaSutra for real court cases?
NyayaSutra is designed for legal research, education, and case preparation. While the AI provides detailed analysis, it should be used as a research tool alongside professional legal judgment. Always consult with qualified legal professionals for actual court proceedings.
My API key shows as "failed". What should I do?
API keys can fail due to rate limits (too many requests) or quota exhaustion. Wait 60 seconds for automatic retry, or click the refresh icon to reset manually. If the problem persists, create a new API key from Google AI Studio.
Are my documents secure?
Your documents are processed locally in your browser and stored in your browser's IndexedDB storage. Document content is sent to Google's Gemini AI for analysis. We recommend not uploading highly sensitive documents and reviewing Google's AI data policies.
Can I edit the AI-generated content?
Yes! You can edit chronology events, key facts, legal theories, and even arguments during simulation. After editing, you can regenerate analysis or continue the simulation with your modifications.
How do I delete a project?
On the Dashboard, hover over a project card and click the trash icon that appears. Confirm the deletion. This action cannot be undone.
What document formats are supported?
NyayaSutra supports PDF (.pdf), Microsoft Word (.docx), and plain text (.txt) files. For best results with PDFs, ensure they contain selectable text (not just scanned images).
Can I use NyayaSutra offline?
No, NyayaSutra requires an internet connection as it uses cloud-based AI services for document analysis and simulation. Your project data is stored locally, but AI features need connectivity.
How do I report bugs or suggest features?
Please report issues and suggestions on our GitHub repository at github.com/ratneshs230/Digital-Courtroom/issues.
